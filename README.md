> Note: This repository contains implementation of skills for agentic coding. For information about the Agent Skills standard, see agentskills.io.

# Skills

Skills are folders of instructions, scripts, and resources that coding agents (Claude, etc.) load dynamically to improve performance on specialized tasks. Skills teach agents how to complete specific tasks in a repeatable way. This repository contains skills for Adaptive Immune Repertoire (AIRR / VDJ-seq) analysis.

## Installation

```bash
npx skills add https://github.com/immunomind/skills --skill airr
```
